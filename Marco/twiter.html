Seleção de Dados

  É um programa do Jupyter que divide tweets pela relevância criado para a diciplina Ciência de Dados do segundo semestre do curso de Engenharia do Insper.

  O programa foi criado usando dados tirados do twiter, que foram classificados de acordo com a relevância a um assunto. Então, criei um programa, que, usando essas classificações, calcula a probabilidade de um novo tweet ser relevante ou irrelevante.

  Ao testar o programa para classificar 200 novos tweets, 87% dos tweets foram classificados corretamente, classificando 96,3% dos tweets irrelevantes corretamente.
